## 网络线路怎么选？

野草云香港VPS有三种主要线路，搞清楚了再下单，别买错了：

**优质BGP**：中国电信回程走CMI，中国联通回程走CU/NTT均衡，中国移动回程Level3。适合大陆用户日常访问，晚高峰偶尔有波动，但整体够用。

**精品BGP**：三网均走骨干网直连（电信回程CU，联通回程CU，移动回程CMI），高峰期更稳定，适合对速度要求高一些的场景，比如跑业务系统、外贸独立站。

**国际BGP**：不含CMI、CU，不做国内优化，内地延迟较高。但带宽大、价格低、DDOS防护强，适合面向东南亚、欧美的跨境电商建站。
